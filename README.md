The evaluation are done on the root mean squared error. RMSE is defined as:
 
$$\left(\frac{1}{N}\sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \right)^{1/2}$$

where
###### $\hat{y}_i$ is the predicted value
###### $y_i$ is the actual value for each instance $i$
